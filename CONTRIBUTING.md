# Contributing

Contributions are welcome.

Good contributions include:

- Better Japanese search terms
- Additional official-source categories
- Improvements to evidence classification
- Better detection of reruns and recycled news
- Documentation and translations
- Tests and bug fixes

Please do not submit:

- Private account content
- Access-control bypasses
- Stalking or private location features
- Unverified accusations presented as fact
- Passwords, cookies, tokens or API keys
