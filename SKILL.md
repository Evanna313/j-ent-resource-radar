# J-Ent Resource Radar Skill

## Purpose

When a user asks whether a Japanese entertainer has new or upcoming work, perform a public-source, receipts-first investigation and clearly distinguish confirmed facts from rumors.

## Scope

Supported topics:

- Television dramas and specials
- Films
- Streaming productions
- Stage productions
- Hosting / MC work
- Major commercials and ambassador roles
- Public information-account spillover
- Public fan reports and filming sightings

## Non-negotiable rules

1. Never treat anonymous information as confirmed.
2. Never bypass login, private accounts, locked posts, paywalls or access controls.
3. Never collect or repeat private location, hotel, flight, home address or contact information.
4. “No public evidence found” does not mean “the artist has no work.”
5. Check whether an item is a rerun, archive clip, old VTR, rebroadcast or recycled news.
6. Preserve the publication date and the date of the claimed future project.
7. Separate evidence from inference.
8. Prefer original announcements over reposts and summaries.

## Mandatory search workflow

### Step 1 — Resolve target and time window

Record:

- Japanese artist name
- Group name
- Common nicknames and aliases
- Current date
- Requested future window
- Requested work types

Do not confuse artists with similar names.

### Step 2 — Tier A: official sources

Search official sources first:

- Talent agency / artist official page
- Television networks
- Program and production official sites
- Film distributors
- Streaming platforms
- Official social accounts
- Press release services

Suggested Japanese queries:

- `<artist> 出演決定`
- `<artist> 主演`
- `<artist> ドラマ`
- `<artist> 映画`
- `<artist> 配信`
- `<artist> 新CM`
- `<artist> 解禁`
- `<artist> 10月期`
- `<artist> 1月期`
- `<artist> 4月期`
- `<artist> 7月期`

### Step 3 — Tier B: reliable named media

Search established entertainment media and newspapers.

For each result, record:

- Publication name
- Publication date
- Whether the article cites a primary source
- Whether it says “confirmed,” “reported,” “expected,” “rumored,” or “according to an insider”
- Whether the article is recycling an older report

Weekly-magazine reporting remains Tier B reporting, not Tier A confirmation.

### Step 4 — Tier C: public SNS

Search public results on:

- X
- Yahoo!リアルタイム
- Threads

Look for:

- Official next-day program previews
- Publicly visible filming notices
- Public fan reports
- Multiple independent reports describing the same event
- Exact screenshots or quotations from a source that can be verified

Do not use follower count as proof of accuracy.

### Step 5 — Tier D: public rumor ecosystems

Search:

- 5ch
- Girls Channel
- Publicly visible information-account spillover
- Public summaries of locked information accounts

Record the exact wording and thread date.

Label all such material as:

- Anonymous forum claim
- Information-account claim
- Second-hand spillover
- Unverified

Never upgrade a rumor merely because many users repeat it.

### Step 6 — Cross-check

For every potentially meaningful claim:

1. Check the original publication date.
2. Search the exact phrase.
3. Search the alleged broadcaster, production company, co-star and time slot.
4. Check whether it matches an existing announced project.
5. Check for extras casting, filming notices or credible production traces.
6. Search for denial, correction or prior failed predictions.
7. Seek a second independent source.

### Step 7 — Assess significance

A meaningful new development usually includes at least one of:

- New official title or casting announcement
- Reliable media report naming a broadcaster, platform, production or role
- Multiple independent public filming reports with corroborating details
- A previously vague rumor gaining a verifiable production trace
- A correction or debunking that materially changes the assessment

Routine fan wishes, old works trending again, magazine reprints and unsupported “soon” hints are not meaningful developments.

## Evidence tiers

- **Tier A — Official confirmation**
- **Tier B — Reliable named media report**
- **Tier C — Public SNS / fan report**
- **Tier D — Information account / anonymous forum**
- **Tier E — Analysis or speculation**

## Required report structure

### 1. Bottom line

State whether there is a meaningful new development.

### 2. Confirmed

List Tier A findings.

### 3. Media reports

List Tier B findings and clearly say they are not official unless confirmed.

### 4. Public SNS and fan reports

List Tier C findings with corroboration status.

### 5. Information accounts and anonymous forums

List Tier D findings with exact caveats.

### 6. Assessment

Explain what the evidence supports and what it does not support.

### 7. What to watch next

List specific official accounts, likely announcement windows or source types.

## Preferred language

Use direct wording:

- “已官宣”
- “可靠媒体报道，但尚未官宣”
- “公开粉丝repo，未获官方确认”
- “信息号/匿名论坛传闻”
- “没有搜到可验证证据”
- “不能由此推导出……”

Avoid:

- “肯定有饼”
- “公司彻底放弃他”
- “业内都知道”
- “这一定是剧宣前兆”

## Motto

**No receipts, no certainty.**
